# 🎓 CodeStack - Online Course Management System

## 📌 Project Description
CodeStack is a Django-based web application that allows users to browse courses, enroll, and learn online. It includes authentication, role-based access, and course management features.

---

## 🚀 Features
- User Registration, Login, Logout
- Role-based system (Admin, Instructor, Student)
- Course creation and management
- Enrollment system
- Notes, Assignments, Live classes
- Payment integration (basic)
- Responsive UI

---

## 🛠 Technologies Used
- Python
- Django
- HTML, CSS, Bootstrap
- SQLite Database

---

## ⚙️ Setup Instructions

1. Clone the repository
   git clone<https://github.com/ummuayisha88-ux/codestack-project>

2. Go to project folder

cd mainproject


3. Install dependencies

pip install -r requirements.txt


4. Run migrations

python manage.py migrate


5. Run server

python manage.py runserver
